September, 2025 Jackfruit Project for Sem 1

Student Helper

Problem: Students often feel crushed under pressure of deadlines, and find it hard to manage time. There arises a need for a handy time and assignment management tool.

Solution: Solution: Built an all-in-one app that helps students manage assignments, deadlines, and submissions with features such as a countdown timer, notes/assignment tracker, calendar to store events, to-do list, and a progress tracker, all using only Python and PyQt.

Takeaways: Learned in depth about PyQt and how to use it along with ample Python practice. This was also my first web-app development project.

💻
2026 AIEP Project

OpenCMS

Problem: Static site content management is manual and fragmented. Existing CMS solutions are often too heavy or not fully Git-native, creating a gap for a lightweight, developer-friendly alternative.

Solution: Developed OpenCMS, it is a minimal headless CMS for publishing blogs directly from a git repository, its built to integrate with hugo, zola, and other static site generators. It provides a markdown editor with live preview and asset uploads. The CMS commits edits back to github automatically, exposes APIs for raw content, metadata, and can trigger site builds through webhooks.

Takeaways: Gained clear understanding of APIs and the working of FastAPI. Learned about over WebDev. First big project that i worked on
