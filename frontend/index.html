👁️
2026 Computer Vision · Healthcare

OpenCMS

Problem: Dermatological conditions are often misdiagnosed in rural India due to a shortage of specialist doctors.

Solution: Developed a CNN-based skin condition classifier using transfer learning (EfficientNetB3), with an explainability layer using Grad-CAM visualizations doctors can trust.

Impact: 91% top-3 accuracy. Submitted for college research paper. Potential for rural clinic deployment.

TensorFlow EfficientNet Grad-CAM OpenCV Flask
GitHub 📄 Paper
03
🎵
2023 Generative AI · Music

MoodMelody

Problem: Music recommendation systems suggest songs based on history, not how you're feeling right now.

Solution: Trained an LSTM model on MIDI sequences + emotion tags, with a real-time mood detection layer via webcam (facial action units). The model generates short melodic loops matching detected mood.

Impact: Won 2nd place at college hackathon. 200+ test users reported high emotional resonance.

LSTM MIDI OpenCV React FastAPI
